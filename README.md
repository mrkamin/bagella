# 👜 Bagella – Bag Ecommerce Store

> Bagella is an ecommerce website, designed to offer shopping experience for high-quality bags across Europe. Customers can browse make collections, filter products, view details, and securely purchase bags from any device.

---
## 🛠  Built With

## Tech Stack

- ![Next.js](https://nextjs.org/)
- ![TypeScript](https://www.typescriptlang.org/)
- ![Tailwind CSS](https://tailwindcss.com/)
- ![Stripe – (for secure payments](https://stripe.com/)
- ![Cloudinary – (for product images / CMS)](https://cloudinary.com/)

---

### ✨ Key Features

- [👜] Browse and filter bags by type, price, and popularity
- [💳] Secure checkout with Stripe integration
- [🧭🔍] Fast and Real-time search, Product Filtering and category exploration
- [📱] Responsive design across desktop and mobile devices
- [💡] Built with performance and SEO in mind

## 💻 Getting Started

To get a local copy up and running, follow these steps.

### ✅ Prerequisites

In order to run this project, you need:

- Node.js installed
- npm or yarn
- Basic knowledge of Next.js and TypeScript

### ⚙️ Setup

1. Clone this repository to your desired folder:
```bash
git clone git@github.com:mrkamin/bagella.git

- Navigate into the cloned folder

```sh
 cd bagella

```

### Install

- Install this project with:

```sh
 npm install
```

- Start the server

```sh
 npm run dev
```
